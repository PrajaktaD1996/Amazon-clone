## Amazon Clone

A static frontend clone of Amazon’s homepage built using HTML and CSS.  
This project focuses on improving UI layout structuring and mastering core frontend fundamentals.

---

## Tech Stack
- HTML5
- CSS3

---

## Features
- Responsive homepage layout inspired by Amazon
- Navigation bar with logo, search bar, and categories
- Product grid sections
- Banner and footer layout design
- Clean UI structuring using HTML & CSS

---

## Purpose / Learning Outcome
- Strengthened understanding of HTML structure
- Improved CSS layout skills (Flexbox, positioning)
- Learned how to recreate real-world UI designs
- Built foundation for advanced frontend development using React

---

## Status
Completed ✔️ (Static UI Project)

## 🔗 Note
This is a frontend-only project focused on UI design. No backend or JavaScript functionality is included.
